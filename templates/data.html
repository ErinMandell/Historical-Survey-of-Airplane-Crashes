Project Data Sources


Geolocated Crash History (1908-2009)

Our primary source is a dataset that is available in CSV format at the Socrata Open Data Network and at Kaggle. The data includes: Crash date, Flight Operator, Passengers Aboard, Fatalities, Location of Crash, as well as a Summary (not all crashes have Summary information). The origin of this data set is unknown, and while it does have 5300+ crash records, it isn't as complete in it's information about more recent years.

We used GoogleMap to geolocate each of these crashes; this complete dataset was then converted to json format. This json data was used to generate the Leaflet maps.

The geolocated crash history data is available to view here
Link to the Kaggle data


Annual Crash Statistics (1918 - 2018)

Our secondary source is the Bureau of Aircraft Accident Archives (BAAA!). This non-governmental organization was established in Geneva in 1990 and has a stated goal of collecting and archiving information relating to avaition accidents worldwide since 1918.

We used this data source to fulfill the scrape requirements of our project as the information here is not available to download. Two srapes were performed - Fatalities by Operator, and Fatalities by Year. These datasets were then converted to json format and were used to generate supporting Plotly charts.
